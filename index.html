Git commands and uses

  1. git add .

    2. tracks new changes to all files

  2. git init

    3. gives git control

  3. git commit

    4. saves changes

  4. git commit -m

    5. saves changes

  5. git push origin main

    6. used to push local git branch to github repository

  6. git clone

    7. clone new repository into new directory

  7. git add

    8. add changes in working directory

  8. git status

    9. show status of changes

  9. git pull

    10. fetch changes from remote repository and merge into new branch

  10. git branch

    11. list create or delete branches

  11. git checkout

    12. switch branches or restore working tree files

  12. git merge

    13. combine changes from different branches

  13. git log

    14. show the commit log

  14. git diff

    15. show changes between commits, commits and working tree e.t.c

  15. git reset - hard HEAD

    16. reset current HEAD to a specified state

  16. git remote -v

    17. manage set of tracked repositories

  17. git fetch origin

    18. download objects and refs from another repository

  18. git stash

    19. stash changes

  19. git tag -m

    20. create, list, delete or verify a tag object

  20. git cherry-pick

    21. apllies changes introduced by some existing commits

  21. git remote add

    22. used to add a remote repository

  22. git rm -r [file-name.txt]

    23. used to remove a file or folder

  23. git branch

    24. enumerates branches

  24. git branch -a

    25. lists all branches both local and remote

  25. git branch [branch name]

    26. creates a branch

  26. git branch -d [branch name]

    27. deletes a branch

  27. git branch -D [branch name]

    28. deletes a branch forcefully

  28. git merge [branch name]

    29. merge a branch into active branch

  29. git stash clear

    30. remove all stashed entries

  30. git push

    31. push changes to remote repository

  31. git revert commitid

    32. revert commit changes

  32. git config --global --list

    33. get global config
